## Project Description

Full Stack App with React.js and Node.js, Express, Sequelize, and Sqlite.

### How to use this app

1. Navigate to the api folder.
2. Run npm install.
3. Run npm start.
4. Navigate to the client folder.
5. Run npm install
6. Run npm start.

Enjoy!
