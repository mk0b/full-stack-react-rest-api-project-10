## Project Description

Create and manage course data with this full stack application.

### Technologies Used:

- JavaScript
- React.js
- Node.js
- Express
- Sequelize
- SQLite3

### How to use this app:

1. Navigate to the api folder.
2. Run npm install.
3. Run npm seed.
4. Run npm start.
5. Navigate to the client folder.
6. Run npm install
7. Run npm start.
8. Navigate to the specified localhost in your browser.
